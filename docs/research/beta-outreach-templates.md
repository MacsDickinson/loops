# Beta Outreach Templates

**Version:** 1.0
**Last Updated:** 2026-03-07
**Owner:** PM
**Purpose:** Recruitment messaging for 20-user beta (Week 3-8 campaigns)

---

## Template Overview

**Goal:** Convert cold/warm contacts to beta applicants

**Channels:**
1. LinkedIn Direct Messages (primary)
2. Email (warm contacts)
3. Community posts (PM communities, AI tool communities)
4. Twitter/X (public announcement)

**Tone:** Direct, empathetic, value-focused (not salesy)

**Key Principles:**
- Lead with pain point, not product features
- Make it personal (no mass spam language)
- Clear CTA (apply for beta)
- Respect their time (<100 words)

---

## 5. Hacker News Post Template

### Template K: Show HN Post

**Title:** Show HN: Discovery Loop Coach – Better Specs for AI Coding

**Body:**

```
Hey HN,

I'm launching Discovery Loop Coach, a tool that helps product managers (and technical founders) write better specs for AI-assisted development.

**The problem:**
AI coding tools like Cursor, GitHub Copilot, and v0 are great, but they often build something different than what you envisioned. The root cause? Vague specs that miss edge cases.

**The solution:**
Discovery Loop Coach uses AI personas (Product, Security, UX, Engineering, Data) to guide you through a structured discovery dialogue. You describe what you want, the AI asks clarifying questions, and you export a precise BDD spec with acceptance criteria.

**How it works:**
1. Start a chat session
2. Describe your feature in plain English
3. Answer questions from AI personas (e.g., "What happens if the user is offline?")
4. Export a markdown spec with Given-When-Then scenarios

**Tech stack:**
- Next.js 16 (App Router)
- React 19
- shadcn/ui (design system)
- Tailwind CSS v4
- Anthropic Claude API (for AI personas)

**Beta details:**
- Free for 3 months
- Limited to 20 users
- Launches in ~2 weeks

**Apply:** [Beta landing page link]

I'd love feedback from the HN community. Happy to answer questions!
```

**Length:** 215 words
**Tone:** Technical, transparent, humble
**CTA:** Landing page link + comment engagement

## Outreach Campaign Schedule

### Week 3-4: Direct Outreach (Goal: 10 signups)

**LinkedIn:**
- Send 10 DMs per week (Template A or B)

- Track: Response rate, application rate

**Email:**
- Send 15 warm emails (Template E)
- Target: Former colleagues, personal network
- Follow-up: Template F after 7 days

**Communities:**
- Post in 5 PM communities (Template G)
- Post in 3 AI tool communities (Template H)

---

### Week 4-6: Content Push (Goal: 5 signups)

**Twitter:**
- Launch announcement thread (Template I)
- Share 3 tweets/week about beta progress
- Engage with replies, quote tweets

**Hacker News:**
- Post "Show HN" in Week 6 (Template K)
- Monitor comments, respond quickly

- Pitch guest post to TLDR, Lenny's Newsletter (Template L)

**Format:** `?utm_source=[source]&utm_medium=[medium]&utm_campaign=beta-launch`

**Examples:**

- LinkedIn DM: `?utm_source=linkedin&utm_medium=dm&utm_campaign=beta-week3`
- Email: `?utm_source=email&utm_medium=warm-outreach&utm_campaign=beta-week3`
- Reddit: `?utm_source=reddit&utm_medium=community-post&utm_campaign=beta-pm-subreddit`
- Twitter: `?utm_source=twitter&utm_medium=thread&utm_campaign=beta-announcement`
